<h3> How to run the code:</h3>
In order to run the code you need VSCode, and G++ Compiler

Open up the repository on github desktop

Choose Open in VSCode

And install any necessary plugins to run C++ such as the 
C++ extension on Github

Then when in the main src code folder

Go to Run>Build without debugging> C++ GDB option
and the code should compile and run!

This will not work in terminal on windows powershell.
